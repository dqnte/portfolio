NANAS

Github

Nanas is the first web app I built with a team. My team consisted of four programmers working for one week to complete the project. Since we were a small team, we decided to adhear to the principles of agile development.

Our team had meeting each morning to discuss the goals of that day. We then sat at the same desk and worked, taking pauses whenever one of us had a feature that involved multiple members.

We finished all the main features of the project early, so we were able to add some extra features. One that I was proud of was the search bar, which can be used from any page. That fun bit with that was integrating it with the filters on the main page, which are only shown on one page.

Another "feature" was the whole website brand. I spent a good amount of time in Photoshop trying to get the right Nana's logo. I should say that the font is Salome, and the banana is (of course) Warhol.
