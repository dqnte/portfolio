PROJECTS

COREO

Coreo (pronounced coreo) is a coreography assistance tool built for dancers. Coreo allows for choreographers to upload videos of 10 second dance routies that their dancers can then practice.

Coreo is a full stack web application built using React+Redux on the front end and Node+PostGreSQL on the backend. Coreo also uses Tensorflow to track the body positions of each dancer's routine.

Nanas

Nanas is an e-commerce website that sells one item: bananas. Users can browse our selection of bananas, add bananas to their cart, checkout, and other miscelenious e-commerce actions.

Nanas was built using React, Redux, Node, and PostGreSQL. Nanas also uses STRIPE for payment checkout and AWS for image hosting. The styles in Nanas are a combination of Sass and Semantic.UI.

Oscars Poll

This is a website I made with a friend for the Academy Awards. My first website was actually made for this exact purpose, so it was fun to revisit the project a year later. The responses are mostly friends and family.

The polling site was built with React, Node.js and Sequelize. It's closed now, but you can take a look if you like. Just submit yor answers at then end to view everyone's results.
